## 🚀 **Features**
- 📍 **Auto Geolocation** - Current location weather  
- 🗺️ **Interactive Map** - Leaflet.js powered
- 📊 **24-Hour Forecast** - Hourly temp + icons
- 🌙 **Dark/Light Mode** - Saved in localStorage
- 📱 **Fully Responsive** - Mobile first design
- ⚡ **No API Key** - Open-Meteo free API
- 🎨 **Smooth Animations** - CSS transitions
